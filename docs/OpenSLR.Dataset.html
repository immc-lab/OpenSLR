OpenSLR.Dataset

Introduction

The Dataset component in OpenSLR is responsible for unified data loading, preprocessing, and multi-stream data generation. It supports various input modalities such as RGB video frames and skeletal keypoints.

Supported Datasets

  • Phoenix14

  • CSL-Daily

  • Custom datasets (via configuration)

Data Structure

Each dataset should be organized as follows:

dataset_root/
├── videos/
├── keypoints/
├── annotations/
│   ├── train.csv
│   ├── dev.csv
│   └── test.csv
└── config.yaml

Usage

from OpenSLR.Dataset import DatasetManager

dataset = DatasetManager.get_dataset("Phoenix14", split="train")
dataloader = dataset.get_dataloader(batch_size=16, shuffle=True)

Preprocessing

  • Frame extraction

  • Keypoint normalization

  • Data augmentation (flip, crop, rotate)

Multi-Stream Support

The Dataset component can handle multiple input streams simultaneously:

dataset:
  streams: ["rgb", "keypoint"]
  rgb:
    resize: [224, 224]
  keypoint:
    normalize: true
